AI-Ready

Root cause, explained by AI.

Click "Analyse with AI" on any failing test. The LLM reads the error message, stack trace, step history, and prior run data — then returns a root cause category, confidence score, plain-English explanation, and a suggested fix. Results are cached by error fingerprint so sibling tests with the same error get instant answers.
